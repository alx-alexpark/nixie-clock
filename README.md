# ИН-12 Nixie Tube Clock

![clock](https://github.com/user-attachments/assets/84f094ff-762f-41dc-9ac6-ecc850881490)

A nixie tube clock made using ИН-12 (IN-12) tubes. Uses k155id1 drivers and a boost converter made using [this reference design](https://github.com/drkmsmithjr/NixiePowerSupply). Time is synchronized over WiFi using NTP (~1sec inaccuracy), and all firmware is written using CircuitPython.
